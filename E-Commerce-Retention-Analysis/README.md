# E-Commerce Customer Retention Analysis

## Project Overview
This project analyses customer behavior in an e-commerce environment to identify opportunities for improving customer retention.

## Objectives
- Identify churn patterns
- Analyse repeat purchase behavior
- Provide strategic recommendations

## Key Insights
- Repeat customers generated higher lifetime value.
- Discount-driven purchases had lower retention rates.
- Customer engagement significantly impacted repeat purchases.

## Recommendations
- Implement targeted loyalty campaigns
- Improve personalised email marketing
- Introduce retention-focused KPIs
